# vite-vue2-template
[![Open in Visual Studio Code](https://open.vscode.dev/badges/open-in-vscode.svg)](https://open.vscode.dev/haodaking/vite-vue2-template)

Vite + Vue2 + Eslint

### Install

```bash
npm install
```
### Development

```bash
npm run dev
```
### Production

```bash
npm run build
```
### Preview

```bash
npm run serve
```
